# Codex Safety Rules

These rules apply to all automated Codex changes.

## Allowed file modifications

Codex may modify files only in these locations:

scraper/parsers/
scraper/core/
scraper/operators/
scripts/
.github/workflows/

Codex may append data to:

data/Unilife_Pricing_Snapshot.xlsx

## Protected locations

Codex must never modify or delete:

.git/
README.md
requirements.txt
data/historical/

## Git safety rules

Codex must never run:

git reset --hard
git clean -fd
git push --force
git rebase

Commits must be normal commits pushed to main.

## Workbook rules

Workbook must remain append-only.

Codex must not:

delete rows
overwrite historical data
create new live workbooks
